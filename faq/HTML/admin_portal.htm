How do I choose my criteria options now that the curriculum criteria menu option is gone?

In order to make the reporting process easier and less time consuming for users we have moved all criteria options to the top of the actual Administrator portal screen.  In order to choose your criteria click on Administrator portal and the criteria options will be available at the top of the screen.  You can now choose and change all of your criteria options from one screen without having to navigate to a totally different menu option each time.

How do I look at specific test types now that the District Assessments and State Assessments option is gone?

In order to distinguish what test type you want to look at you can use the Assessment Category criteria from the Administrator Portal screen.  Please see the answer from the question above for a further explanation of the new Administrator Portal screen.
Where did all the reports go (i.e. At Risk Standards by Student, by Class, by Teacher etc..)?  I only see a couple under each cell I click.

All At Risk, Mastery, Progress and Report Card reports have been consolidated into dynamic reports that can be broken down into all the old reports.  All of the different At Risk reports are now located in the left nav after you run the 'At Risk' report.  Once you run the initial report you can break it down by different criteria by clicking through the reports in the left nav.  This same structure is used in the Mastery, Progress and Report Card reports.
What does the icon next to teacher's names do?

These icons allow permitted users direct access to the Teacher Portal.  Administrator no longer have to navigate away from the Admin portal to run the Teacher Portal, they can just click the icon to link directly to the associated teacher's teacher portal.
Can I print reports with page breaks so I can hand them out to students?

Yes, on many of the various reports you run, including the At Risk and Mastery reports you can run the report with page breaks.  The page break options are located in the left nav once the report has been ran under the With Page Break header.
How do I compare different test types and categories?  Is there a way to get them all to show on the same screen?

New in the Summer 2010 release is a locking feature that allows the user to compare multiple test types and assessment categories.  Once you run a report you will see an "unlocked" icon above the test name.  Click this icon to "Lock" the test to your admin portal.  Once a test has been locked you change all of you criteria and search for another test.  The "Locked" test(s) will then display on your newly ran report next to the other tests.
Can I filter the Administrator Portal by student demographics?  If so, how?

Yes, in the top right of your Administrator Portal there is a new Demographics section that allows the user to break down their Admin portal reports by demographic.
Can I look at state data through the Administrator Portal?  If so, how?

Yes, in the top left of the Administrator Portal there is an Assessment Category drop down.  If state data is loaded to your system this drop down will allow you to view state data results.
What do the different "clickable" cells represent?

Each of the cells represents a different combination of tests, classes, teachers and students.  For example, the "paper" icons will always run reports on ALL tests for either the district, school, teacher or class depending on which row the user is on.  The corresponding row to the column determines what entities are included in the report.  If you do not wish to run a report on ALL tests you can click the actual percentage scores below one of the columns for a specific test.
Can I print the Administrator Portal view?

Yes, if you scroll over the Actions menu above the portal you can click the Print option to print the report.
Can the Administrator Portal be extracted to Excel?

Yes, if you scroll over the Actions menu above the portal you can click the Export to Excel option to export the report.
