What kind of information can I see when searching for state standards?

When searching for standards you will be able to see State Standards broken down by Strand, Standard, and Element (terminology will change dependant on state). You will also be able to see the standard names and descriptions. Please note that if the description seems too short or is cut off you can hover-over the description and you’ll be able to read the whole thing.

 

You can also see how many items are associated with each standard and view those items by clicking on the number in the right column. You can also add an item with that standard tied to it by clicking the plus sign under the ? Add column. This functionality will only be available to certain user types depending on district permissions.
How can I see how many items are tied to a standard?

When you search for standards on the right is a column named “? Count”. If you expand the standard set there will be a value next to each strand showing the question count.  You can also click on this number to view the actual items.  This functionality will only be available to certain user types depending on district permissions.
What is outline view?

Outline view provides an expandable and simplified view of the standards.  In order for the outline view to display all criteria must be filled in: set, grade, subject, course and standards.

Is there a quick way to add items for standards in the State Standards Search page?

Yes! Next to each standard in the State Standards Search page is a green plus sign under the ? Add column. If you click that button you will be taken to the item builder screen with the selected item already attached to the standard.
