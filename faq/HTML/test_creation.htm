How do I change the term for a newly created test?

In order to change the term on a newly created test the grade, subject and course must be selected and saved for the term drop down to show up.

 

Why does my standards drop down for a question say "none"?

New in the Elements Summer 2009 Release the Standard Search Options now defaults to Standard Filter.  If you are seeing "none" in the standard drop down it means that you currently do not have a filter built.  You can either build a Standards Filter using the Filter option in the left nav of the test page, or under the Curriculum tab, or you can change the Standard Search Options to Search or Standards Outline.

 

In the Standard Search Options for a test, what is the Standards Outline?

The Standards Outline is a new standard search option that was added with the Elements Summer 2009 Release.  The Standards Outline automatically populates with all the standards associated with the grade, subject and course tied to the test.  When using the Standards Outline you can either hover over the standard name to see the standard text or click the preview icon to view the standard itself.

 

How do I change the order of the questions on a test?

In order to change the order of questions on test you can change the Item # next to the question in the left column of the content page.  You will need to change both questions that are being switched.  For example you can change the item # for question 2 to 1 and the item # for question 1 to 2 and hit save.

 

How do I remove a question from a test?

In order to completely remove a question from a test you will need to click the red X next to the Item # in the left column of the question.  The red X will change to a green check mark and a new menu option will appear in the left nav called Remove Selected that once clicked will remove the checked questions.

 

What is the difference between removing a question and erasing the content?

Removing a question using the red X will remove the entire question from the content and the total item count for the test will change.  Erasing the content using the Erase Question button only deletes the information for that question but leaves the blank question.  It does not change the total item count.

 

Why, when I use the search questions option, does it leave blank questions ony my test?

The search questions feature adds a new question to the test.  It does not change the content of a blank question on the test.  For example if a test has 10 questions and the search questions feature is used it will add an eleventh question to the test, not change one of the existing blank questions.

 

What information can I change on a test after it has been proofed?

After a test has been proofed there are only a few components that can be changed.  The standards, correct distractor, score on test checkbox and security tab are the only areas that can be changed after a test is proofed.
