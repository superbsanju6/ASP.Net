How does a teacher control the criteria for the teacher portal?

The teacher's view of the teacher portal is completely dependant on the criteria they choose in the two drop downs in their left nav.  If a teacher wishes to view a different class in the teacher portal all they have to do is change their course and curriculum drop downs in the left nav.
Can I print reports with page breaks so I can hand them out to students?

Yes, on many of the various reports you run, including the At Risk and Mastery reports you can run the report with page breaks.  The page break options are located in the left nav once the report has been ran under the With Page Break header.
How do I compare different test types and categories?  Is there a way to get them all to show on the same screen?

New in the Summer 2010 release is a locking feature that allows the user to compare multiple test types and assessment categories.  Once you run a report you will see an "unlocked" icon above the test name.  Click this icon to "Lock" the test to your admin portal.  Once a test has been locked you change all of you criteria and search for another test.  The "Locked" test(s) will then display on your newly ran report next to the other tests.
Can I filter the Teacher Portal by student demographics?  If so, how?

Yes, in the top right of your Teacher Portal there is a new Demographics section that allows the user to break down their Teacher portal reports by demographic.
What do the different "clickable" cells represent?

Each of the cells represents a different combination of tests, classes, teachers and students.  For example, the "paper" icons will always run reports on ALL tests for either the teacher, class or student depending on which row the user is on.  The corresponding row to the column determines what entities are included in the report.  If you do not wish to run a report on ALL tests you can click the actual percentage scores below one of the columns for a specific test.
Can I print the Teacher Portal view?

Yes, if you scroll over the Actions menu above the portal you can click the Print option to print the report.
Can the Teacher Portal be extracted to Excel?

Yes, if you scroll over the Actions menu above the portal you can click the Export to Excel option to export the report.
