What State Data files can be uploaded?

Elements currently support the following state data files:

CRCT

EOCT

GHSGT

GHSWT

ITBS

FCAT

05 Writing

08 Writing

 

This list will be updated as more state data files are supported.
What format does State Data need to be sent in?

State data files need to be sent in as text (.txt)  or data (.dat) files.  In order for the upload to work correctly the files must be submitted in this format.  If a file is not submitted as a .txt or .dat file the upload will be unable to process and Thinkgate Support will request the document be sent again.
How do I send a State Data file to Thinkgate?

In order to send a State Data File to Thinkgate Support you must have tmanager rights in your system.  If you are a tmanager, listed below are directions:

 

1.Click Support
2.Click Upload Files to Support
3.Click Browse in the resulting pop-up
4.Located the file on your computer
5.Double click the file once located
6.Click Continue
How long does it take to get a State Data file uploaded?

Once a state data file has been received by Thinkgate Support, minus any import issues, the file will be uploaded within two weeks.  If there are any import issues with the file the import process could take longer.  If the issues cannot be resolved by Thinkgate Support you will be contacted any may be required to re-send the file after changes are made.  If an issue is found in the import and Thinkgate Support makes the request for a new file or correction to the existing file the two week time period will start again as soon as the corrections are made or new file received.
