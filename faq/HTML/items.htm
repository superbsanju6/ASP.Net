How do I create new items in Elements?

To create a new item in Elements, follow these steps:

 

1.Click Assessment
2.Click the Add Items icon
3.Enter the desired criteria for the Item on the Profile tab
4.Select the Standards tab, enter search criteria and attach the desired standards to the Item
5.Select the Content tab and edit the question, edit the answer options and select the correct answer in the answer drop down.  You can also run spell check on the item
6.Click Save
7.Select the Preview tab to view the item

 

Explanations of the different items banks can be found in the FAQ below.
When creating items, there are different Item bank options that can be selected.   What are the differences in the item banks?

In Elements there are 5 different item banks.  Below are the different item banks and an explanation of each.

 

1.Personal Bank- The Personal Bank is each user’s private bank.  When a user creates an item it will automatically be saved to this bank.  Other users will not be able to use items from this bank unless the user shares the item.
2.School Bank- When a user shares an item through school bank, the item will be shared among all teachers in that particular school.
3.Teacher Bank- When a user shares an item through the teacher bank the item will be shared among all teachers in the district.
4.District Bank-The district bank is shared among users in the District Office.  Once an item is designated for the District Bank it does not reside in any other bank.
5.Thinkgate Community Bank-The Thinkgate Community Bank is comprised of approved content and customer items promoted from proofed district assessments.  This bank is shared by all Thinkgate customers.
Can I copy an item from the teacher or school bank?

Yes, you can copy items from other banks to save them into your own.  You can use the item as is or you can edit the item to your liking.  In order to copy an item, follow these steps:

 

1.Click Assessment
2.Click the Find Items Icon
3.Enter the search criteria and click find items, be sure to select which item bank you would like to search for the item in
4.Once you find the desired item, click the copy question icon under item status

 

The item is now copied into your personal item bank and can be found and edited by searching for items in your personal bank
How do I search for items from a specific item bank?

In order to search for items in a particular bank follow these steps:

 

1.Click Assessment
2.Click the Find Items Icon
3.Choose the criteria to search for the desired question
4.Change the item bank drop down to the bank you would like to search
5.Click Find Items.  Only Items from the bank you have selected will appear
How do I add an item with an addendum?

In order to add an addendum to an item, follow these steps:

 

1.Click Assessment
2.Click the Add Items Icon
3.Select the Profile Tab
4.Enter the desired item criteria
5.Click the Standards tab
6.Attach the desired standards
7.Click the Content Tab
8.Enter desired item and answer content and select the correct answer
9.Click the Addendum Icon

 

At this point there are two different options. You can either add a new addendum by selecting Add Addendum or find an existing one by selecting Find Addendums.

 

Add Addendum Option:

 

1.Click Add Addendum
2.Enter the addendum criteria on the Profile tab
3.Click Save
4.Select the Content Tab
5.Enter the content for the addendum
6.Click Save

 

Find Addendum Option:

 

1.Click Add Addendum
2.Enter the Search criteria
3.Click Find Addendums
4.Click the Select Icon next to the addendum you want to choose
5.Click Save
