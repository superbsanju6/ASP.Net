Data Imports and Loads

 

How do I upload a roster file?

A roster file needs to be uploaded to Thinkgate support through the Upload to Support option under the Support tab.  The file needs to be sent as a comma delimited or .csv (comma-separated values) file.

 

How do I upload an Admin or School file?

An Administrator or School file is uploaded the same way that a Roster File is uploaded, through the Upload to Support option under the Support tab.  These files need to be sent as comma delimited or .csv (comma-separated values) files.

 

How long does it take to get a roster file uploaded?

Roster files should be uploaded within 24 hours of being received by Thinkgate support unless there are any issues during the upload process.  Roster files sent late Friday will be uploaded at the beginning of the following week.

 

How do I upload state data?

State data is uploaded to the Thinkgate support the same way a Roster File is, through the Upload Files to Support option located under the Support tab. However, state data should be sent as a text or .txt file.

 

How long does it take to get state data uploaded after it has been sent?

Once state data has been uploaded to and received by support there is a two week turn around to get the data uploaded to the respective system. The two week time frame is subject to change in the case of any upload or technical issues.

 

Why is the birthdate and state ID required in a Roster File?

The birthdate and state ID are required in a Roster File because State Data imports are matched to specific students using the birthdate and state ID.
