What Student Response Systems are supported by the Elements™  Clicker Software?

Currently supported clickers are listed below:

 

1.eInstruction® - RF, IR and Pulse
2.Renaissance Clickers

 

We are currently researching and developing support for additional clickers.  This list will be updated as more clickers are supported.
How do I obtain the Elements™ Clicker Software?

In order to obtain the Elements clicker software please contact your district administrator or Thinkgate Support (if permitted) to receive the software.
Do I need to install anything else besides the Elements™ Clicker Software?

If you are using the eInstruction® IR, eInstruction® RF, or Renaissance clickers you will not need to install any additional software. However, if you are using the eInstruction® Pulse clickers you will need to install the CPS software that is located on the eInstruction software.
How do I administer a assessment using the Elements™  Clicker Software?

In order to administer an assessment utilizing the Elements clicker software, please follow the steps below:

 

1.Verify proper software is installed (Elements Clicker Software and CPS Software if using Pulse clickers)
2.Verify Receiver is properly plugged in
3.Open the Elements Clicker Software
4.Enter in your user ID and password (same one used to access Elements)
5.Click Log In
6.Enter the Test ID into the Test ID field (the test ID is located on the dashboard for the test event)
7.Click Load Test
8.Enter the appropriate Pad #'s in the Pad # column for each student (click Enter after typing in the number)
9.Once a student's clicker information is entered their clicker will auto receive the test
10.Once the clicker is ready the student can proceed through the test
11.Click Finished once all results are entered to submit scores
