What is a Standards Alignment document?

A standards Alignment document is a supplemental document that can be attached to a standard to provide the user with further information and assistance with teaching the standard.  A standards alignment allows the user to fill in the Essential Learning, Essential Vocabulary, Assessment and Resources for a specific standard and all the "sub-elements" associated with the standard.  This information is then stored in an electronic document in the Elements™ system that is attached to a standard.
How do I search for a Standards Alignment document by Standard?

In order to search for a Standards Alignment document please see the directions listed below:

 

1.Click on Instruction
2.Click the Find Icon next to Standards Alignment under the Instructional Planning section
3.On the right side of the search criteria click Standards (it will appear as the only option that is not grayed out)
4.This will open a new set of criteria you can use to pick the Standard Set, Grade, Subject and Standards
How do I add a Standards Alignment document?

In order to add a Standards Alignment document please see the directions listed below: (Only certain user types have access to add Standards Alignment documents)

 

1.Click on Instruction
2.Click the Add Icon next to Standards Alignment under the Instructional Planning section
3.Click Add Standards Alignment
4.A new Document will open on the Document - Profile screen, fill in a description if desired
5.Click Associations in the left nav
6.Click the green plus sign to add standards to the document
7.A new pop up will open that will allow you to search for standards using the criteria.  Search for the desired standards and check the Sel box next to the standards desired.
8.Click Update Standards in the left nav
9.Click Save
10.Click Associations in the left nav and the standards will now be listed
11.Click Content in the left nav
12.You will now see your desired standards listed as well as every subordinate element for that standard
13.You can now use the Click to Edit boxes to enter in information for the different sections
14.Click Save periodically as you complete the document
When I add a Standards Alignment I see the following error on the Content tab, "Error: Required data is missing for the following Associations: Standards".  What does this error message mean?

If you are seeing this error message it means that your document does not have any standards tied to it yet.  In order to begin editing the document standards must first be attached.  Please follow these steps to add standards to the standards alignment (the steps start assuming you already have the document open):

 

1.Click Associations in the left nav
2.Click the green plus sign to add standards to the document
3.A new pop up will open that will allow you to search for standards using the criteria.  Search for the desired standards and check the Sel box next to the standards desired.
4.Click Update Standards in the left nav
5.Click Save
Can I paste tables or other external content into the text editor?

We highly recommend that you do NOT paste any external content into the Click to Edit text editors.  Content from external sources often contains html tags that can cause issues when trying to view the document later.  We recommend typing any of your data into the editor.
