How do I add a new student to the system?

In order to add a new student to the system please follow the directions below.   Only certain user types have access to add new student, if you do not see any of the menu options listed in the directions below please contact your district administrator:

 

1.Click Support
2.Click the Add Student Icon
3.Enter Student Info on the Profile Page
4.Select Demographics in the left nav and enter demographic information
5.Click Save

 

Prior to adding a student you will want to make sure they do not already exist in the system.  Use the Find Students icon to search for the student prior to adding them.  Make sure you change the Enrollment drop down to “All Years” when performing the search.
How do I edit an existing student?

In order to edit a student please follow the directions below. Only certain user types have access to edit students, if you do not see any of the menu options listed in the directions below please contact your district administrator:

 

1.Click Support
2.Click the Find Student Icon
3.Use the criteria to search for the desired student
4.Select Find Students (If searching for new students who are not currently tied to classes, change the enrollment drop down to all years)
5.Select Find Students
6.Click the Edit Icon next to the student that needs to be updated
7.Make changes to the preferred information
8.Click Save

 

If you are changing a student's ID please make sure your change is reflected in your Student Information System or the Roster Load could cause the creation of duplicate students.
When I try to add a student it will not let me set their student id, why?

If you are unable to set a Student ID, more than likely the ID is already in the system.  Check to be sure that the ID for this student is correct.  Also check the system for users with the ID.  Change the Enrollment Drop down to All Years to ensure that you are searching for all students in the system. Use only the student ID criteria to search for the student, leave all the other fields blank or on "All".
Why can't I find the student I just added to the system?

In order to search for a recently added student the enrollment drop down in the search criteria will need to be set to all years.  Students who are not currently tied to any classes will not show up under current year.

 

1.Click Support
2.Click the Find Student Icon
3.Use the criteria to search for the student, be sure to change the enrollment drop down to All Years
4.Select Find Students
How do I delete a student from the system?

In order to delete a student from the system please follow the steps below. If a student has any scores, current or prior, tied to them you will be unable to delete them.

 

1.Click Support
2.Click the Find Students Icon
3.Use the criteria to search for the desired student
4.Click Find Students
5.Click the Edit Icon next to the student to be deleted
6.Click Delete in the left nav
How do I enroll a student in a class? (Enrollment Utility directions included)

There are two ways a student can be enrolled in a class.  One is through the class itself and the second is by utilizing the Enrollment Utility. The enrollment utility is only available to certain user type based on district permissions.

 

Option 1:  Through the class, follow the directions below:

1.Click Support
2.Click the Find Icon for Classes
3.Use the criteria to search for the desired class
4.Click Find Classes
5.Click the Edit Icon next to the desired class
6.Select the Students tab in the left nav
7.Click Add Students
8.Use the criteria to search for the student (If the student is not already in current classes you will need to change the Enrollment drop down to "All Years")
9.Click Find Students
10.Check the box next to the student being enrolled
11.Click Select Students
12.Click Return to Class

 

Option 2: Student Enrollment Utility

1.Click Support
2.Under the Students section click Enroll Students
3.Use the criteria to search for the desired student
4.Click Find Students
5.Select the Add checkbox next to the desired student (multiple students can be selected)
6.Click the Students Clipboard icon (located in the bottom right of the Students criteria box)
7.Click Classes (located at the top of the criteria box)
8.Use the criteria to search for the desired class
9.Click Find Classes
10.Select the Add checkbox next to the desired class (multiple classes can be selected)
11.Click the Classes Clipboard icon (located in the bottom right of the Classes criteria box)
12.Click Enroll
13.If the Class Conflicts pop up displays you can click Remove to move the student from their old class to the new one or you can click Retain to keep them in both classes

 

For more detailed instructions and lessons on the Enrollment Utility please visit your reference center to view the Enrollment Utility documentation and hands on lessons.
How do I remove a student from a class?

There are two ways to remove a student from a class, through the Class screen or through the Student screen. Both of these options are only accessible by specific user types depending on district permissions.

 

Option 1:  Through the Student Screen

1.Click Support
2.Click the Find Students Icon
3.Use the criteria to search for the desired student
4.Click Find Students
5.Click the Edit Icon for the desired student
6.Select the Schedule Button
7.Select the Remove Icon for the class the student needs to be removed from
8.Click Save

 

Option 2: Through the Class Screen

1.Click Support
2.Click the Find Classes icon
3.Use the criteria to search for the desired class
4.Click the Edit icon for the desired class
5.Select the Students tab in the left nav
6.Check the check box next to the desired student in the roster
7.Click Remove Students
What do I do if a student is duplicated in the system?

If you find a duplicate student in the system you can first try to delete one of the student records by following the steps in the "How do I delete a student from the system" FAQ.  If the system allows you to delete the student it means you deleted the duplicate with no data tied to them.  However, if the system will not allow you to delete either one of the student records it means both have data associated with them.  You will need to contact your district administrator, or Thinkgate Support (if permitted), in order to get the student records merged.
