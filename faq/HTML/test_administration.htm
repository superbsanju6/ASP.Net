Dashboard

 

Why is the dashboard grayed out?

If the dashboard for a test is grayed out the test security is more than likely set to Inactive.  When the test security is set to inactive the dashboard will be grayed and not viewable until a date is specified within the Test Security window. This can be changed through the security tab under the test profile screen for a proofed test.  The status can also be changed through the Find District Test screen by selecting the checkbox next to the correct test and then using the Actions hover over menu to change the status.

 

Test security can only be changed by specific user types within your district.  Please contact your district's Thinkgate Administrator in order to find out who can change the security settings of a test.

 

How do I reset a student's score?

In order to reset a student's test score you will need to check the select checkbox next to the student's name and click the Reset Selected option in the left nav.

 

How do I manually input a student's score?

In order to manually input a student's score you will need to check the select checkbox next to the student's name then click the Manual Input Selected option in the left nav which will open a separate window that will allow you to manually enter question answers for the student. This is most commonly used to enter short answer question scores.
