How do I add a new school to the system?

In order to add a new school to the system please follow the directions below. Only certain user types have access to add new schools, if you do not see any of the menu options listed in the directions below please contact your district administrator:

 

1.Click Support
2.Select the Add Icon next to Schools
3.Fill in the appropriate information:

School Name

Abbreviation (optional)

Phone Number (optional)

School Type

Cluster (optional)

School ID

4.Click Save

 

The school ID needs to match the school ID that exists in your Student Information system if a roster load for the school is ever desired.
Do I need to add a cluster to my school?

Depending on how your district organizes schools you may want to add a cluster flag to your school.  Clusters allow for larger district to group their schools.  Please check with your district office to find out if this is required.
How do I edit a school?

In order to edit a school please follow the directions below.

 

1.Click Support
2.Click the Find Icon next to schools
3.Use the criteria options to search for the school you wish to edit.
4.Click Find Schools
5.Click the Edit icon next to the desired school
6.Once the profile page for the School loads any information can be edited on this screen.
7.Click Save
