How do I add an Image to Elements?

In order to add a new image to Elements follow these steps:

 

1.Click Assessment
2.Click the Add Images Icon
3.Enter the desired criteria for the image
4.Click Browse
5.Find the image on your computer and double click
6.Click upload
7.Click Save
How do I add an Image to an Item?

In order to add an image to an item, follow these steps:

 

1.Navigate to the desired item using the Find Item icon under the Assessment button
2.Click the content tab for the item
3.Select click to edit
4.Select the Insert Image icon
5.Select Find Images or Upload new Image

 

Find Image Option:

 

1.Enter the desired search criteria
2.Click Find Images
3.Click the Select Icon next to the image to be selected
4.Click Save

 

Upload New Image Option:

 

1.Select Upload New Image
2.Enter the criteria for the new image
3.Click Browse
4.Find the image on your computer and double click
5.Click upload
6.Click Save
