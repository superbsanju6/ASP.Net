Scanning

 

How will I know that everything scanned properly?

In order to verify that the bubble sheets were scanned properly you can check the dashboard for the test that was scanned and make sure that every student has a score showing up under the Pct Score column.  Another way to verify that the sheets were scanned correctly is by checking the Manual Input for a student and comparing it to the original bubble sheet.  In order to do this go to the dashboard for the test and click the checkbox next to the students name and click Manual Input Selected in the left nav.

 

See also How do I manually input a student's score? for directions on accessing the Manual Input screen

 

Can anyone login to scan bubble sheets?

Thinkgate support works with district administrators in order to determine who has permission to scan bubble sheets within a district.  All Elements users have the ability scan bubble sheets, however, Thinkgate support recommends that district administrators control scanning permissions within their district.  Please contact your district's Elements™ administrator for scanning permission.

 

Can one person scan for multiple teachers?

An individual user does have the ability to scan for multiple teachers, however, permission to do so is controlled by a district's Elements™ administrator.  Please contact your district's Elements™ administrator in regard to the scanning permissions within your district.

 

How does the scanner respond to duplicate student ID's?

If multiple bubble sheets are scanned with the same student ID the first bubble sheet scanned for the student is the only one that is actually scored.  All the other sheets scanned with the same student ID will be ignored unless the student's score is reset through the dashboard and the sheets scanned again.  The first sheet scanned for a student ID is always the one that is scored.

 

Should the sheet be face up or face down?

Both HALO and Plain Paper bubble sheets are scanned face up.

 

Where can I get the PPS scanning software?

The Thinkgate PPS scanning software is located in the reference center within the Elements™ system.  The Thinkgate scanning software must be downloaded in order to scan tests within the Elements systems.

 

With the 2009 summer release the old PPS software will be automatically updated to the new version when the software is opened the first time.

 

I administered the test on the wrong bubble sheet, so how do I rescan the bubble sheets for the right test?

If the test was taken on either a student or HALO form you can print off a blank form for the correct test and scan that form on top of the other forms.  When the blank form is scanned first it will scan all the other forms to the test id on the first bubble sheet.
If the tests were scanned on a roster form the test will need to be taken over again or the scores manually entered through the test dashboard.

 

Student scores should be reset through the dashboard for the incorrect test that was scanned

 

How do I switch scanning modes?

In order to change the scanning mode on the scanner hold ctrl + shift + F12 with the scanning software open.

 

What happens to test results if bubble sheets are scanned for a class that already had results?

If bubble sheets are scanned for a class that already had test results the new bubble sheets scanned will be ignored and the old scores will remain.  In order to scan new bubble sheets for a class that already has results the student's scores must be reset.  In order to do this go to the dashboard and select the checkboxes next to the desired students then click Reset Selected in the left nav.  After the scores have been reset the new bubble sheets can be scanned.

 

Why are no scores showing on the Dashboard after I scanned the test?

The most common reason for scores not showing on the dashboard after a test was scanned is because the test contained a short answer or essay question.  Scores for these questions must be manually input for the students in order for the scores to be calculated.
