Can new performance level sets be added?

In order to add a Performance Level Set, follow these steps. New Performance Level Sets can be added by higher end users only

 

1.Click Support
2.Click the Find Performance Levels Icon
3.Choose a Year from he drop down and enter a Name
4.Click Add Set
5.The newly added Performance Level Set will display.  Click the Set Levels Icon under the Levels column.  One level will be added by default, click Add Level to add more.
6.Enter in the Level Name, Abbreviation, Min Score, Max Score and Color for each of your levels.
7.Click Save

 

Once a performance level set is added to the system it is available for use on tests. All assessments will still default to the DistrictDefault performance set and must be manually changed if you desire to use the new one.  Follow the steps in "Is it possible to have assessment use different performance level sets" FAQ for details on how to change the performance level set for an assessment.
Can existing performance level sets be edited?

Yes, existing performance levels can be edited.  In order to edit a performance level follow these steps. Only higher end users can edit performance level sets.

 

1.Click Support
2.Click the Find Performance Levels Icon
3.Select the year and Name if known
4.Click Find Performance Level Sets
5.Click the Set Levels Icon under the Levels column
6.Make the desired changes to the Performance Level Set levels.
7.Click Save

 

If the District Default performance level set is edited it will reflect the changes on ALL assessments within your system.  Please exercise caution when editing Performance Level Sets due to the impact it will have on reporting.
Is it possible to have assessments use different performance level sets?

Yes, it is possible to have different assessments use different performance level sets.  In order to do this you will need to manually set the performance level set you want the assessment to adhere to, by default, all assessments are tied to the District Defaults performance level set.  You must have custom added performance level sets in order to change the set on the assessment.

 

Here are some steps to change the performance level set on an assessment:

 

1.Navigate to the profile of an "unproofed" assessment (District or Classroom) by using the Find Icons for District or Classroom under the Assessment button.
2.On the profile page locate the Scoring section and the Performance Level Set drop down
3.Use this drop down to change the performance level set for the assessment

 

The Performance Level Set for an assessment cannot be changed after a test is proofed.
