Quick Start Guides and Webinars

 

Where can I find the Quick Start Guides?

The Quick Start Guides can be found in the Reference Center under the Support tab.  Quick Start Guides are documents designed to give thorough step by step directions on individual functions and components of the Elements system.

 

Where can I find the webinar schedule/information?

The webinar schedule and information can be found under Upcoming Events in the Support Center located within the Support tab.  Webinars are web based meetings that allow the participant to talk to and view the computer screen of a Thinkgate employee for a more in depth overview and training.
