Item Builder: Items, Images and Addendums

 

Can I upload test questions from Microsoft Word?

Currently there is not a way to upload questions from an external source.  Questions must be manually entered into the item bank using the Add Item feature.

 

Are there any restrictions on image sizes?

The suggested size for images uploaded to the system is 300x300 dots per inch (dpi) for two column tests.  Images on one column tests can be as large as 600x600 dpi.

 

Can I add questions to Elements that come directly from a text book?

If a question is added from an external source such as a text book the copyright information must be filled in.  The copyright drop down on the Item Profile screen will need to be changed to 'Yes' and the credit field filled in with the proper information for the original source of the question.
