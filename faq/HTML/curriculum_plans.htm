Pacing Guides

 

Can I copy a pacing guide and use it for the next school year?

Yes, a pacing guide can be copied and used for the next year.  In order to do this you will need to use the Copy Content From option on the new pacing guide. Go to the pacing guide for the new school year and there will be an option in the left nav, under the Actions section, called Copy Content From. This option will open a new window that will allow you to search for old pacing guides to copy to the new one, you just need to select the correct plan and click the Add Curriculum Plan button.
