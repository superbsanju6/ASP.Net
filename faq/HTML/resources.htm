What kind of resources can I add to Elements™?

There are currently two different types of resources that can be added to Elements.  The two types of resources are web link resources and uploaded document resources.  The web link resources allows the user to link a web page directly to a resource stored in Elements.  The uploaded document resource allows the user to upload an external document to store in a resource bank in Elements.
What is the purpose of adding  a resource?

The purpose of adding resources is to provide supplemental document and websites that can assist with the teaching of a particular standard.  The resources can be tied to a specific standard then accessed by an user who may be teaching that standard.
How do I add a resource?

In order to add a resource, please follow these steps:

 

Upload Document Resource option:

 

1.Click Instruction
2.Click the Add Resources icon
3.Select the template name
4.Click Add Resource
5.Add a Description on the Profile Tab
6.Click the Associations Tab
7.If you desire associate standards to the resource click the Add icon
8.Search for and add the desired standard(s)
9.Click the Content Tab
10.Click Replace File
11.Click Browse
12.Select the desired document from your computer and click Upload
13.Click Save

 

Web Link Option

 

1.Click Instruction
2.Click the Add Resources icon
3.Select the template name
4.Click Add Resource
5.Add a Description on the Profile Tab
6.Click the Associations Tab
7.If you desire associate standards to the resource click the Add icon
8.Search for and add the desired standard(s)
9.Click the Content Tab
10.Type or paste a web link into the Web Link bar at the bottom of the screen (the address should start with http://)
11.Click Replace Link
12.Click Save
What can I associate the resources to?

Standards can be associated to a resource to provide supplemental material for teaching a particular standard.

 

In order to associate a resource to a standard, please follow these steps:

 

1.Navigate to the desired resource through the Instruction tab and the Find Resources icon
2.Click the Associations tab on the Resource Profile Page
3.Click the Add Standards Icon
4.Enter the desired search criteria
5.Click Search
6.When you find the correct standard click the Add icon
7.Click Save
Do I have to associate a resource with something?

You do not have to associate a Resource with anything but can associate them with a standard if desired.
