Accessing Elements
After entering my user name, password and client ID, then clicking Sign In, nothing happens.  Why am I unable to access Elements?

In order for Elements™ to open correctly, pop-up blockers must be turned off in your internet browser.  If you never see the system open after signing in through the www.thinkgate.net homepage, more than likely, pop-up blockers are enabled in your internet browser.  In order to turn off pop-up blockers, here are some directions for Internet Explorer 7, Internet Explorer 8 and Mozilla Firefox (3.0+):

 

Internet Explorer 7:

1.Click Tools
2.Hover over Pop-Up Blockers
3.Click Turn Off Pop-Up Blocker

 

Internet Explorer 8:

1.Click Tools
2.Hover over Pop-Up Blockers
3.Click Turn Off Pop-Up Blocker

 

Mozilla Firefox (3.0+):

1.Click Tools
2.Click Options
3.Click Content
4.Uncheck Block Pop-Up Windows
5.Click OK

 

Once you have disabled pop-up blockers sign into Elements™ through www.thinkgate.net.  A pop-up should display after you enter your User Name, Password and Client ID.
The pop-up blockers for my internet browser have been turned off, however, I am still unable to access Elements.  Why am I unable to access Elements?

If Elements™ is not opening, even after turning off the pop-up blockers in your internet browser, than there is probably a 3rd party add-on installed for your internet browser that contains a separate pop-up blocker.  Normally, any kind of tool bar add-on could cause this issue.  You will need to disable these third party add-ons for the system to open properly.  Here are some directions for Internet Explorer 7, Internet Explorer 8 and Mozilla Firefox (3.0+):

 

Internet Explorer 7

1.Click Tools
2.Hover over Manage Add-ons
3.Click Enable or Disable Add-ons...
4.Search the resulting list for any toolbar add-ons
5.Click the desired add-on you wish to turn off
6.Under the Settings section select the Disable radio button
7.Click OK
8.You may need to repeat this for any other add-on that has a pop-up blocker
9.You may need to close and re-open your browser once you have completed these steps

 

Internet Explorer 8

1.Click Tools
2.Click Manage Add-ons
3.Click Toolbars and Extensions
4.Search the resulting list for any toolbar add-ons
5.Click the desired add-on you wish to turn off
6.Click Disable at the bottom right of the screen
7.Click OK
8.You may need to repeat this for any other add-on that has a pop-up blocker
9.You may need to close and re-open your browser once you have completed these steps

 

Mozilla Firefox (3.0+)

1.Click Tools
2.Click Add-ons
3.Click Extensions (check Plugins if you do not see any under Extensions)
4.Search the resulting list for any toolbar add-ons
5.Click the desired add-on you wish to turn off
6.Click Disable
7.Close the Window
8.You may need to repeat this for any other add-on that has a pop-up blocker
9.You may need to close and re-open your browser once you have completed these steps

 

Printing Reports
How do I display the performance level colors and bar graph colors on printed reports?

In order to print the performance level set colors on a report (or any other colors), the background colors and images setting must be turned on in your internet browser.  Below are directions on how to turn this setting on for Internet Explorer 7, Internet Explorer 8 and Mozilla Firefox (3.0+):

 

Internet Explorer 7:

1.Click Tools
2.Click Internet Options
3.Click Advanced
4.In the list under settings scroll down to the Printing section
5.Check the checkbox next to “Print Background Colors and Images”
6.Click OK

 

Internet Explorer 8:

1.Click File
2.Click Page Setup
3.Check the checkbox next to “Print Background Colors and Images”
4.Click OK

 

Mozilla Firefox (3.0+):

1.Click File
2.Click Page Setup
3.Check the checkbox next to “Print Background (colors and images)”
4.Click OK

 

Printing Assessments
When printing assessments that contain page breaks the pages are not printing correctly.  The breaks are incorrect.  What do I need to do to make the page breaks print correctly?

In order for the page breaks on assessments to print out correctly (through the one column or two column print) the proper settings must be set for Headers and Footers in your internet browser.  Listed below are some directions on changing the Header and Footer settings in your internet browser:

 

Internet Explorer 7:

1.Click File
2.Click Page Setup
3.Erase anything entered in the Header and Footer fields (they should both be blank)
4.Click OK

 

Internet Explorer 8:

1.Click File
2.Click Page Setup
3.Set each drop down in the Headers and Footers section to “Empty”
4.Click OK

 

Mozilla Firefox (3.0+)

1.Click File
2.Click Page Setup
3.Click Margins & Header/Footer
4.Set each drop down in the Headers & Footers section to “—blank—“
5.Click OK
What are the recommended page margin settings to print assessments?

In order for tests to print out correctly we recommend you set specific page margins in your internet browser.  Listed below are some directions on setting the proper page margins in your internet browser.

 

Internet Explorer 7:

1.Click File
2.Click Page Setup
3.Under the Margins (inches) section set the left, right, top and bottom margins to “0.5”
4.Click OK

 

Internet Explorer 8:

1.Click File
2.Click Page Setup
3.Under the Margins (inches) section set the left, right, top and bottom margins to “0.5”
4.Click OK

 

Mozilla Firefox (3.0+):

1.Click File
2.Click Page Setup
3.Click Margins & Header/Footer
4.Under the Margins (inches) section set the left, right, top and bottom margins to “0.5”
5.Click OK

 

Pre-Slugging HALO Forms
When pre-slugging a HALO form, the student ID and test ID are displaying "on top" of the bubbles.  How do I make the student ID and test ID print in the correct boxes?

In order to properly print pre-slugged HALO forms, the page scaling in your internet browser must be turned off.  Listed below are some directions on how to turn off page scaling in your internet browser.

 

Internet Explorer 7:

1.Click File
2.Click Print
3.Change the Page Scaling drop down to “None”
4.Click OK

 

Internet Explorer 8:

1.Click File
2.Click Print
3.Change the Page Scaling drop down to “None”
4.Click OK

 

Mozilla Firefox (3.0+)

1.Click File
2.Click Print
3.Change the Page Scaling drop down to “None”
4.Click OK

 

Internet Explorer 8 - NOMERGE
In rare cases, users signing in are seeing more menu options than they should.  Also, some users search results are not consistent with the restrictions set for their user type.  How do I correct this issue?

If you are using Internet Explorer 8 and having multiple users logging in on the same computer, or one user logging in as multiple different user types, you will need to add the "-nomerge" statement to your Internet Explorer program.

 

1.Right Click the Internet Explorer 8 icon
2.Click Properties
3.Click Shortcut
4.In the Target field enter ‘ – nomerge’ after .exe” (at the end of the target field).  Make sure to include the blank space before –nomerge.
5.Click OK
