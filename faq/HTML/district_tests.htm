Can I proof multiple district tests in a term?

Yes, as long as each test is a different test type.  Only one test type can be proofed per term unless the Exam test type is used.
I would like to copy one of our Pre Tests and make it a Post Test.  Is this Possible?

Yes, in order to copy a test here are some directions:

 

1.Click Assessment
2.Click the Find District Assessment Icon
3.Search for the desired test using the drop downs
4.Click Find District Assessments
5.Click Edit next to the desired assessment
6.Click the Copy Test button in the left nav
7.Once the copy has been created change the test type
8.Click Save

 

Copying a test will create a second test with the exact same content.  Once the copy has been created, any of the profile information can be changed and saved.
I have administered a test and scanned the bubble sheets.  Is it possible to reset a student’s score after tests have been scanned?

Yes, it is possible to reset a student's score after the bubble sheets have been scanned.  In order to reset a student score here are some steps:

 

1.Click Assessment
2.Click the District Assessment Dashboard icon
3.Search for the desired test (use the test id if available, if not search by teacher name, class, test type and term)
4.Click Find Test Events
5.Click the Dashboard Icon for the desired test event
6.Click the checkbox next to the student(s) that need to be reset
7.Click Reset Selected in the left nav

 

Resetting a student's results will completely clear their results from the system.  The only way to recover the scores is to re-scan or manually input the score again.
I have created and proofed a test.  How can I target this test to schools or teachers?

Once a district test is created and proofed it is automatically targeted to every teachers that has the particular grades/subject/course available to them that the test was created as.  A district test does not need to be targeted in the same way a classroom test is.
How do you activate the security status of a test so the teacher can click the dashboard icon?

When test security status is set to disabled Elements users are unable to access the test dashboard in order to administer the test.  Customers may set this status in order to ensure that tests are administered during a specific time period.  In order to activate the test security, follow these steps:

 

1.Click Assessment
2.Click the Find District Assessment Icon
3.Enter proper search criteria
4.Click Find Assessments
5.Click the Edit Icon for the desired assessment
6.Click the Security Tab
7.Change the drop down to enabled
8.Click Save

 

The Assessment dashboards are now accessible and tests may be administered.
How do I enable online testing?

In order to enable online testing here are some directions:

 

1.Click Assessment
2.Click the District Assessment Dashboard icon
3.Search for the desired test (use the test id if available, if not search by teacher name, class, test type and term)
4.Click Find Test Events
5.Click the Dashboard Icon for the desired test event
6.Click the Enable/Disable Test option in the left nav

 

This will enable or disable the online testing depending on the current status of the test.
I have administered a test and realized after words that one of the answers on the test is incorrect.  Can I change a test answer and if so, how will the score be affected?

Yes, a correct answer choice can be changed even after a test has been proofed and scanned.  In order to change the answer choice here are some steps:

 

1.Click Assessment
2.Click the Find District Assessment Icon
3.Search for the desired test using the drop downs
4.Click Find District Assessments
5.Click Edit next to the desired assessment
6.Click Content in the left nav
7.Using the correct answer drop down change the distractor for the question desired
8.Click Save (this save could take longer than a normal save due to the changes being made)

 

Once an answer choice has been changed all scanned results will be automatically adjusted to mirror the change.
What can I change on an assessment after it has been proofed?

There are 3 main elements that can be changed after a test has been proofed:

 

1.Correct Distractor choice for a question
2.Questions can be removed from scoring
3.Standards attached to questions
What does it mean when the content on a test is disabled?

When test content is disabled users are unable to see questions on the test.  This means that users are unable to view the test its self and also that no links are available to view the questions through reports.  For example if the content on a test is disabled a user can run a distractor analysis and can see what question numbers students missed but they will not have a link to view the question content as they do when content is not disabled.

If a student retakes a test, will that score override the original score?

In order for a student's score to be replaced they must first be reset.  If a new bubble sheet is scanned when a score already exists it will not overwrite their current score.
