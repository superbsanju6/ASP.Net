What is a Standards Filter?

A Standards Filter is a pre determined list of standards that a user can create to make free form Assessment construction much faster.  A Standards Filter is unique to every user and will "stick" with a user until deleted.  After building a Standards Filter the user will have access to their filter through the Standard Filter option under the Standard Search Options on an assessment.  When Standard Filter is chosen the standards from the filter will display as choices for each question allowing the user to have a much shorter list to choose standards from for the assessment.
How do I build a Standard filter?

In order to build a standards filter please follow the directions below:

 

1.Click Curriculum
2.Click Filter
3.Select the criteria for the standard(s) you would like in your filter
4.Click Find Standards
5.Select the check boxes under  the Sel column next to the desired standards
6.Click Update Filter

 

To remove standards from the filter:

1.Uncheck the box next to the standard to be removed
2.Click Update Filter

 

A standard filter can be updated or deleted at any time.
