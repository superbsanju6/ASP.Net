What do I need to administer an assessment with Plain Paper Bubble Sheets?

You will need to select a form type to administer the test with (roster or student forms). If you choose roster forms you can print one from Elements™ and make a photocopy for each student. If you choose student forms you will need to print one for each student. Both form types will have the test ID printed on them.

 

To print bubble sheets follow these steps:

 

1.Log into Elements™
2.Click on Assessment
3.Click on Find (Classroom or District) Test Events (the dashboard icon)
4.Search for the desired test
5.Click on the Dashboard Button
6.Place check marks next to the desired students (or hover over Actions and click Select All)
7.Click the Bubble Sheet Button
8.Click Print Student Form
Or
Click Print Roster Form

 

You will also need a blank copy of the test for the student to look at. This can be obtained on the profile page of the test. Please follow these steps to print a blank copy of a test:

 

1.Log into Elements™
2.Click on Assessment
3.Click on Find (Classroom or District) Assessments
4.Search for the desired test
5.Click the Edit Test Button
6.Click on Blank Test (1 column), Blank Test (2 column) or Blank Test (Word)
7.Print the desired blank test
What are the different kinds of Plain Paper Bubble Sheets and what is the difference?

There are three kinds of bubble sheets available in Elements™; student, roster and blank.

 

1.A student form is unique to the student. It comes with the student ID and test ID pre-printed on it.
2.A roster form includes all students in the class and the student bubbles in his or her own name. The primary benefit of a roster bubble sheet is that one can be printed and then photocopied to save printer ink.
3.A blank form is a bubble sheet with the test ID on it but no student information filled in. The student’s name can be filled in manually. The blank form is used to administer a test to a student who may not be in the class roster. The bubble sheet can be scanned at a later date after the student has been enrolled.
