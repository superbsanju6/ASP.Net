How do I add a new class to the system?

In order to add a new class to the system please follow the directions below.

 

1.Click Support
2.Click the Add Classes Icon
3.Enter the information on the profile screen
4.Click Save

 

Once you have saved the profile information for the class you have added a "shell" for the class.  The next step is to add teachers and students.  Adding teachers and students to a class are both covered in the next two FAQ, please follow from step 6 to the end for both "How do I add teachers to a class" and "How do I add students to a class".
How do I add teachers to a class?

In order to add teachers to a class please follow the steps below.  If you are adding teachers to an already created class please start at step 1.  If you have just added the class and are on the profile screen please start at step 6.

 

1.Click Support
2.Click the Find Class Icon
3.Use the criteria to search for the desired class.
4.Click Find Classes
5.Select the Edit Icon for the desired class
6.Select Teachers
7.Click Add Teachers
8.Enter the Teacher Name, school etc. to narrow the search results
9.Click Find Teachers
10.Check the box next to the desired teacher(s)
11.When the teacher has been chosen Click Select Teachers
12.Click Return to Class

 

Multiple teachers can be added to a class.  The first teacher added to a class will be marked as the primary teacher, any additional teacher will be treated as a secondary.  The Reset Primary option on the Teachers tab of the class will allow you to change the primary teacher if desired.
How do I add students to a class?

In order to add students to class please follow the steps below.  If you are adding students to an already created class please start at step 1.  If you have just added the class and are on the profile screen please start at step 6.

 

1.Click Support
2.Click the Find Class Icon
3.Use the criteria to search for the desired class
4.Click Find Classes
5.Select the Edit Icon for the class the student(s) will be added to
6.Select Students
7.Click Add Students
8.Enter criteria to find the  desired student(s)
9.Click Find Students
10.Check the box next to the desired student(s) Multiple students can be selected
11.Click Select Students
12.Click Return to Class
How do I delete a class?

In order to delete a class please follow the directions listed below:

 

1.Click Support
2.Click the Find Classes Icon
3.Use the Criteria to find the class to be deleted
4.Click Find Classes
5.Click the Edit Icon next to the class to be deleted

 

In order to completely delete a class the Students and Teachers must be removed first.

 

6.Click Teachers in the left nav
7.Hover over Actions and select Check All
8.Click Remove Teachers
9.Click Students in the left nav
10.Hover over Actions and select Check All
11.Click Remove Students
12.Click Profile in the left nav
13.Click Delete Class
What does "Retain on Resync" mean?

If a class is marked as "Retain on Resync" it means that the next roster load will not delete the class.  If this option is not checked for a class, the class will be removed when the next roster file is imported.  We strongly recommend making sure this box is checked if you do not want the class to be deleted during the next import.  If the class is deleted it will need to be manually added back after the import.
How do I find the assessments available to a class?

Test events for a particular class can be viewed from within the class "edit" screen.  In order to locate and view these test events please follow the directions below:

 

1.Click Support
2.Click the Find Classes Icon
3.Use the criteria to search for the desired class
4.Click Find Classes
5.Choose the Edit Icon next to the desired class
6.Select Test Events from the Class Profile Screen

 

The Test Events tab will display all assessments available to the class.  If you wish to administer one of these assessments click the dashboard for the desired event.
