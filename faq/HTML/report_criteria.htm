Why can't I click the question links on the Item Analysis?

If the individual item links are not clickable within the item analysis report it is more than likely due to the fact that the test security is set to content – Disabled.  This can be changed through the administration tab under the test profile screen for the individual test.  The content status can also be changed through the Find District Test screen by selecting the checkbox next to the correct test and then using the Actions hover over menu to change the status.

 

Where can I learn about all the different reports offered in the Elements system?

In the Reference Center within the Elements™ system there is a powerpoint labeled Elements Reports Definitions that contains a detailed description and explanation of all the different reports offered within Elements™.

 

Why are none of my results showing up in the report engine after the test has been taken and scored?

The most common reason for scores not showing up within the report engine is due to the fact that the test was administered through a Homeroom class rather than the individual class for the respective subject.  This is most common in Elementary classes but it depends on how tests are administered within your district.  Try changing your report criteria with Homeroom as the subject then use the course drop down to find the correct test.
