What is the recommended procedure for administering a test online?

In order to administer an assessment online we recommend navigating to the assessment dashboard on one computer (if possible) while your students log into the testing site on their computers.  The main reason you want to have one computer on the dashboard for the test event is so that you can unlock students if they become locked out as well as easily let the students know their student ID and test ID.  Please follow the directions below for both the administrator of the test and the students.

 

For the person administering the test:

1.Click Assessment
2.Click the Dashboard Icon next to District or Classroom depending on test
3.Use the criteria to search for the desired test event
4.Click the Dashboard icon for the test event
5.This screen will display all the information you need to administer the test

 

For the students:

1.Students will need to navigate to the following address: www.thinkgate.net/<insert client ID here>/test, make sure you enter your client ID where asked in the URL.
2.Student will then be prompted for their student ID and test ID which can be located on the dashboard for the test

 

For further instruction on administering a test online please refer to the training material located in the Reference Center under Support.
When I try to administer a test online none of the students can log into the test.  Why can't they login?

There are a couple things to check if a student is unable to sign into a assessment.  Please verify the situations listed below if students are having issues logging in:

 

1.Verify they are using the correct Student ID and Test ID (located on the dashboard)
2.Verify the Online Status for the test is Enabled (listed at the top of the dashboard screen).  If not, click Enable/Disable in the left nav of the dashboard.
3.Verify there is not a score or started date already present for the student (visible on the dashboard).  You will need to reset the score or date if the student is showing a score or started date.
4.If none of the above work it is possible there is a duplicate login ID's in the system or a caching issue.  If none of the above solve the issue please contact your district administrator or Thinkgate Support (if permitted) to get the issue resolved.
How does lock and unlock work?

The Locking feature on an Online Test prevents a student from searching external sources on their computer.  If a student clicks anywhere outside the testing window they will be locked out of the test until the administrator of the test unlocks them. We recommend that the student does not close the testing window if they become locked out.
How do I enable or disable online testing for an assessment?

Online testing can be enabled through the dashboard for the test event.  Please follow the directions below:

 

1.Click Assessment
2.Click the Dashboard Icon next to District or Classroom depending on test
3.Use the criteria to search for the desired test event
4.Click the Dashboard icon for the test event
5.Use the Enable/Disable option in the left nav to change the online status of the test (listed at the top of the dashboard)
If a student does not finish a test is there a way to allow them to continue the assessment at another time?

Yes, it is possible to suspend/continue an assessment for a student.  Please follow the directions below:

 

1.Click Assessment
2.Click the Dashboard Icon next to District or Classroom depending on test
3.Use the criteria to search for the desired test event
4.Click the Dashboard icon for the test event
5.Check the checkbox next to the desired student
6.Click Suspend Selected in the left nav
7.In order to allow the student to continue the test click Continue Selected in the left nav after selecting the student.
