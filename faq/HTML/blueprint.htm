How do I add a new blueprint?

A blueprint cannot be "added" to the system, only the shell edited.  In the beginning of a new semester Thinkgate will add a blank "shell" for every Grade/Subject/Course combination in the system.  Users can then edit these shells.  The reason you cannot add a new blueprint is due to the issues that could be causes by a course having multiple blueprints.
Why is there no blueprint available to a specific subject and course?

There should be a Blueprint in the system for each subject and course. If one appears to be missing please contact Thinkgate Support or your district point of contact.
How do I edit a blueprint?
1.In order to edit a blueprint please follow the directions listed below.

 

1.Click on Assessment
2.Click on the Find Blueprint Button
3.Use the criteria to search for the desired blueprint
4.Click Find Blueprints
5.Click the Edit Button for desired blueprint
6.After clicking Edit you will be brought to the Profile page of the blueprint

 

For detailed instruction on how to Edit and Create a Blueprint please view the Blueprint lessons within your Reference Center.  These guides will provide step by step instruction and images on how to edit a Blueprint.
What are the different parts of a blueprint?

Listed below are descriptions of each of the different sections within a Blueprint:

 

1.Standards – State Standards can be added to the Blueprint to ensure that all of the standards for that grade, subject and course are a covered in assessments.
 
2.Test Plan – The test plan is where the user decides what kinds of assessments will be given in which term.
 
3.Test Schedule – This page mirrors the selections made on the Test Plan page. On the Test Schedule the user can change the sequence of the assessments being given within each term. The user can also mark the Blueprint Required check box to make the corresponding assessments required to follow the blueprint format.
 
4.Blueprint – The Blueprint page gives the user the ability to select how many questions from each standard will be on each assessment.
 
5.Assessment Guide – Provides a high-level overview of the Blueprint.

 

For detailed instruction on how to Edit and Create a Blueprint please view the Blueprint lessons within your Reference Center.  These guides will provide step by step instruction and images on how to edit a Blueprint.
Once I create a blueprint do I have to use it for every assessment I create?

If the Blueprint Required checkbox is selected on the Test Schedule portion of the Blueprint then the corresponding assessment will make the test conform to the Blueprint.  This checkbox can be "unchecked" at any time.  If you wish to create a free form test instead of using the blueprint you can uncheck the blueprint required box on the Test Schedule tab of the blueprint.
Why do I need to proof my blueprint?

Once the Blueprint is proofed, users will be able to build assessment off of it.
What can I change after a blueprint is proofed?

After a blueprint has been proofed anything can be updated.  Proofing a blueprint only allows assessment to be built off of it.  It does not lock any of the editable fields.
What is the difference between a blueprint and a pacing guide?

A Blueprint is a schedule of what the students are going to be tested on. A Pacing Guide is a schedule of what is going to be taught.
What are the green highlighted boxes when I look at standards?

The green boxes reflect the Pacing Guide and show what standards are scheduled to be taught in each term.
